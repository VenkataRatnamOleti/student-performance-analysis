# Student Performance Analysis System

A simple Python project to manage student records, analyze academic performance, and visualize results using the console and CSV files.

## Features

- Add, display, update, delete student records
- Search students by ID
- Calculate averages, top scores
- Save/load records to/from CSV file
- Visualize subject scores with bar charts
- Modular code built with functions and classes

## Setup Instructions

1. Install Python (3.7+ recommended)
2. Install required Package:
    `pip install matplotlib`
3. clone or copy all source files to a folder
4. Create an empty CSV file at `data/students.csv` if not present

## How to Run

In a terminal:
    `python main.py`

Follow on-screen menu to add, view, update, analyze, and visualize student data.

## Notes

- Data is saved in `data/students.csv`
- Console clears before each menu for better readability
- Visualization uses Matplotlib (pop-up window)
- Works cross-platform (Windows, Mac, Linux)
